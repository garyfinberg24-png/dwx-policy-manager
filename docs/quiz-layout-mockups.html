DESIGN NOTES Quiz Builder Start Screen

What's New

  • KPI Summary Bar — Total quizzes, published count, drafts, average pass rate
  • Policy Link — Each card shows the linked policy (mandatory field)
  • Pass Rate Stat — Live metric from quiz results
  • Quick Actions — Edit, Preview, Duplicate on every card
  • Scheduled Progress — Shows date range and progress bar for scheduled quizzes
  • Archived Visual — Reduced opacity + "Delete" action instead of "Edit"
  • Empty State — "AI Generate from Policy" secondary action for first quiz

Design Decisions

  • Cards over Table — Quizzes have rich metadata; cards show more at a glance
  • 4px border-radius — On all controls (buttons, search, filter tabs)
  • 8px border-radius — On cards and containers
  • 40px content padding — Matches global layout update
  • Forest Teal (#0d9488) — Primary actions, active states, links
  • Grid: minmax(340px, 1fr) — Responsive, fills width, 2-3 columns on desktop
  • Card hover — Subtle lift + shadow on mouse over

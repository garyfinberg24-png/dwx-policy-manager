Containment Actions

Golden Rule: Contain first, investigate second. Stop the bleeding before performing forensics.

Immediate Containment (First 30 Minutes)

  • Isolate affected systems from the network (do NOT power off — preserve evidence)
  • Disable compromised user accounts
  • Block malicious IPs, domains, and file hashes at firewall/EDR
  • Revoke active sessions and force MFA re-authentication
  • Activate incident bridge (Teams: "Incident Response" channel)

Extended Containment

  • Deploy temporary monitoring rules for lateral movement detection
  • Snapshot affected VMs for forensic analysis
  • Engage external IR retainer if P1 (CrowdStrike: contract #FD-IR-2026)

Communication & Notification

Internal Escalation (All Severities)

  • P1: CISO + CTO + CEO + Legal within 30 minutes. Board notified within 4 hours.
  • P2: CISO + CTO within 1 hour. CEO if data involved.
  • P3-P4: IT Security Manager within 4 hours. CISO in daily summary.

External Notification Requirements

  • GDPR: Supervisory authority within 72 hours of confirmed personal data breach
  • OAIC: Australian Privacy Act — Notifiable Data Breach scheme (eligible data breach)
  • Clients: Contractual notification obligations per agreement (typically 48 hours)
  • Law enforcement: When criminal activity is suspected (coordinate with Legal)

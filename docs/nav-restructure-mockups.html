Design Decisions
  • Vertical separator between flat items and dropdown groups
  • Chevron ▾ indicates dropdown — rotates on hover/open
  • Sub-item descriptions — secondary text in dropdown for clarity
  • Dropdown closes on click outside, Escape key, or item selection
  • Active child highlights parent dropdown (e.g., if on Analytics, Manager is teal)
  • 4px border-radius on dropdown panel, consistent with global style
  • Reports is new — maps to PolicyManagerView (team compliance)
  • Executive Dashboard is new — maps to a summary/KPI view

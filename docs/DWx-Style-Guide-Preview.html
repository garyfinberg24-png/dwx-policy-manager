3. Font Weights

Strict font weight hierarchy. Default to 400, use 500 for emphasis, NEVER use 600.

⚠️ Critical Rule
Never use font-weight: 600 (semibold). It creates visual inconsistency and looks "muddy" between normal and bold.

Weight Comparison

400 (Normal) - Default for body text, labels, field values
font-weight: 400 | --dwx-font-weight-normal
500 (Medium) - Titles, buttons, badges, table headers
font-weight: 500 | --dwx-font-weight-medium
600 (Semibold) - DO NOT USE THIS WEIGHT
font-weight: 600 | PROHIBITED
700 (Bold) - KPI numbers and statistics ONLY
font-weight: 700 | --dwx-font-weight-bold

9. KPI / Stats Cards

Standardized KPI/Stats card styles for dashboards. Stats Card Style 1 (Label First + Pill Trend) is the DEFAULT - use unless an alternate style is specifically requested.

Official Stats Card Styles (January 2026)
Style 1: Label First + Pill Trend is the official DEFAULT stats card style.
Style 2: Number First + Pill Trend is the official ALTERNATE style.
Both use pill-shaped trend badges for change indicators. See detailed specifications below.

Stats Card Style 1: Label First + Pill Trend (DEFAULT)

The official default implementation. Label appears first to provide context, followed by the large number value and optional pill-shaped trend badge. Best for dashboard overviews where users need to quickly understand what each metric represents.

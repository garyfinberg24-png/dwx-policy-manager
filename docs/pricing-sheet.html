Cost Optimisation Notes
  • Azure Free Tier: Application Insights (5 GB/month), Function Apps (1M executions/month), and Storage (5 GB) all include generous free tiers that cover typical usage.
  • Reserved Instances: Not applicable — all resources use Consumption/Pay-As-You-Go tiers with no minimum commitment.
  • AI Toggle: AI features (Quiz Generator, Chat Assistant) can be disabled in Admin > AI Assistant settings. When disabled, Azure OpenAI cost = $0.
  • Logic App: Can be paused in Azure Portal when not needed (e.g. during non-business hours) to reduce polling costs.
  • Region: Sweden Central for AI (data residency), Australia East for email/distribution (latency to SP tenant).
